Платформы и подключение

Включите платформы и следуйте инструкциям для подключения автопубликации

💬 Telegram
Как подключить:
1. Открой @BotFather в Telegram
2. Отправь /newbot → придумай имя → получишь токен
3. Создай канал и добавь бота как администратора (право "Post Messages")
4. Вставь токен и @username канала ниже
🎮 Discord
Как подключить:
1. Открой сервер Discord → Settings → Integrations → Webhooks
2. Нажми New Webhook → выбери канал
3. Нажми Copy Webhook URL → вставь ниже
🐦 Twitter / X
Как подключить:
1. developer.x.com → Create App → OAuth 1.0a (Read and Write)
2. Вставь все 4 ключа ниже
💼 LinkedIn
Как подключить:
1. linkedin.com/developers → Create App → запроси w_member_social
2. Получи Access Token и Person URN (urn:li:person:XXXXX)
📘 Facebook
Как подключить:
1. developers.facebook.com → Create App
2. Получи Page Access Token и Page ID
📷 Instagram
Как подключить:
1. Instagram → Settings → Switch to Professional (Business) account
2. Подключи Instagram к Facebook Page (Settings → Linked Accounts)
3. Graph API Explorer → permissions: instagram_basic, instagram_content_publish
4. GET /me/accounts → найди page → GET /{page_id}?fields=instagram_business_account
🧵 Threads
Как подключить:
1. Тот же Meta Developer App что для Instagram
2. Permissions: threads_basic, threads_content_publish
3. Threads User ID = твой Instagram/Threads user ID
